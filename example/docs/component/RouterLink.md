# RouterLink

`<RouterLink>` is the component for enabling user navigation in a router-enabled app.

## Props

### to

* type: `string`
* default: `'/'`

```html
<RouterLink to="/home">Home</RouterLink>
```

### replace

* type: `boolean`
* default: `false`

### className

* type: `string`
* default: `''`

### activeClassName

* type: `string`
* default: `'router-link-active'`

### match

* type: `string`
* default: `''`
